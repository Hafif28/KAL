SPL 3 VARIABEL (ELIMINASI GAUSIAN)¶

Contoh SPL 3 Variabel¶

Kita akan menyelesaikan sistem persamaan berikut:

{1x+1y+1z=62x+3y+1z=101x+2y+3z=13\begin{cases} 1x + 1y + 1z = 6 \\ 2x + 3y + 1z = 10 \\ 1x + 2y + 3z = 13 \end{cases}⎩⎨⎧​1x+1y+1z=62x+3y+1z=101x+2y+3z=13​
(1)

Langkah-langkah Eliminasi Gauss (Proses Detail):

  1. Representasi Matriks Augmented: $$

    [111∣6231∣10123∣13]\begin{bmatrix} 1 & 1 & 1 & | & 6 \\ 2 & 3 & 1 & | & 10 \\ 1 & 2 & 3 & | & 13 \end{bmatrix}⎣⎡​121​132​113​∣∣∣​61013​⎦⎤​
    (2)

    $$

  2. Langkah 1: Normalisasi Baris 1

    • Karena A11A_{11}A11​ sudah 1, tidak ada perubahan: R1→R1R_1 \to R_1R1​→R1​

  3. Langkah 2: Eliminasi Elemen di Bawah Pivot Kolom 1

    • Baris 2: R2=R2−2R1  ⟹  [2,3,1,10]−2[1,1,1,6]=[0,1,−1,−2]R_2 = R_2 - 2R_1 \implies [2, 3, 1, 10] - 2[1, 1, 1, 6] = [0, 1, -1, -2]R2​=R2​−2R1​⟹[2,3,1,10]−2[1,1,1,6]=[0,1,−1,−2]

    • Baris 3: R3=R3−1R1  ⟹  [1,2,3,13]−1[1,1,1,6]=[0,1,2,7]R_3 = R_3 - 1R_1 \implies [1, 2, 3, 13] - 1[1, 1, 1, 6] = [0, 1, 2, 7]R3​=R3​−1R1​⟹[1,2,3,13]−1[1,1,1,6]=[0,1,2,7]

  4. Langkah 3: Normalisasi Baris 2

    • Karena A22A_{22}A22​ sudah 1, tidak ada perubahan: R2→R2R_2 \to R_2R2​→R2​

  5. Langkah 4: Eliminasi Elemen di Bawah Pivot Kolom 2

    • Baris 3: R3=R3−1R2  ⟹  [0,1,2,7]−1[0,1,−1,−2]=[0,0,3,9]R_3 = R_3 - 1R_2 \implies [0, 1, 2, 7] - 1[0, 1, -1, -2] = [0, 0, 3, 9]R3​=R3​−1R2​⟹[0,1,2,7]−1[0,1,−1,−2]=[0,0,3,9]

  6. Langkah 5: Normalisasi Baris 3

    • R3=13R3  ⟹  13[0,0,3,9]=[0,0,1,3]R_3 = \frac{1}{3}R_3 \implies \frac{1}{3}[0, 0, 3, 9] = [0, 0, 1, 3]R3​=31​R3​⟹31​[0,0,3,9]=[0,0,1,3]

  7. Hasil Akhir (Bentuk REF): $$

    [111∣601−1∣−2001∣3]\begin{bmatrix} 1 & 1 & 1 & | & 6 \\ 0 & 1 & -1 & | & -2 \\ 0 & 0 & 1 & | & 3 \end{bmatrix}⎣⎡​100​110​1−11​∣∣∣​6−23​⎦⎤​
    (3)

    $Setelahinikitalakukan∗BackSubstitution∗untukmendapatkannilai Setelah ini kita lakukan *Back Substitution* untuk mendapatkan nilai Setelahinikitalakukan∗BackSubstitution∗untukmendapatkannilaix, y, z$.

import numpy as np

# 1. Definisikan Matriks Augmented
A = np.array([
    [1, 1, 1, 6],
    [2, 3, 1, 10],
    [1, 2, 3, 13]
], dtype=float)

print("Matriks Awal (Augmented Matrix):")
print(A)
print("-" * 30)

# 2. Proses Eliminasi Gaussian dengan Cetak Langkah per Langkah
n = len(A)

for i in range(n):
    # Membuat pivot baris i kolom i menjadi 1 (Normalisasi)
    pivot = A[i][i]
    if pivot != 0:
        A[i] = A[i] / pivot
        if pivot != 1:
            print(f"Langkah {i+1}.1: Normalisasi Baris {i+1} (R{i+1} / {pivot:.2f})")
            print(A)
    
    # Menolkan elemen di bawah pivot
    for j in range(i + 1, n):
        factor = A[j][i]
        if factor != 0:
            A[j] = A[j] - factor * A[i]
            print(f"Langkah {i+1}.{j-i+1}: Eliminasi Baris {j+1} (R{j+1} - {factor:.2f} * R{i+1})")
            print(A)
    print("-" * 30)

print("\nMatriks setelah Eliminasi (Bentuk REF):")
print(A)

# 3. Substitusi Balik (Back Substitution)
x = np.zeros(n)
for i in range(n - 1, -1, -1):
    x[i] = A[i][-1] - np.sum(A[i][i+1:n] * x[i+1:n])

print("\nSolusi Sistem:")
print(f"x = {x[0]:.2f}")
print(f"y = {x[1]:.2f}")
print(f"z = {x[2]:.2f}")

SPL 4 VARIABEL (ELIMINASI GAUSIAN)¶

Sistem Persamaan Linear 4 Variabel (SPL 4V)¶

Kita akan menggunakan sistem yang memiliki solusi unik agar baris terakhir tidak menjadi nol:

{2x1+3x2−x3+4x4=11x1−2x2+2x3−3x4=23x1+1x2+1x3+2x4=3−1x1+4x2+1x3−2x4=4\begin{cases} 2x_1 + 3x_2 - x_3 + 4x_4 = 1 \\ 1x_1 - 2x_2 + 2x_3 - 3x_4 = 2 \\ 3x_1 + 1x_2 + 1x_3 + 2x_4 = 3 \\ -1x_1 + 4x_2 + 1x_3 - 2x_4 = 4 \end{cases}⎩⎨⎧​2x1​+3x2​−x3​+4x4​=11x1​−2x2​+2x3​−3x4​=23x1​+1x2​+1x3​+2x4​=3−1x1​+4x2​+1x3​−2x4​=4​
(4)

Proses Detail Operasi Baris Elementer:

1. Matriks Augmented Awal¶

[23−14∣11−22−3∣23112∣3−141−2∣4]\begin{bmatrix} 2 & 3 & -1 & 4 & | & 1 \\ 1 & -2 & 2 & -3 & | & 2 \\ 3 & 1 & 1 & 2 & | & 3 \\ -1 & 4 & 1 & -2 & | & 4 \end{bmatrix}⎣⎡​213−1​3−214​−1211​4−32−2​∣∣∣∣​1234​⎦⎤​
(5)

2. Kolom 1 (Pivot R1)¶

  • Normalisasi R1: R1=R12  ⟹  [1,1.5,−0.5,2,0.5]R_1 = \frac{R_1}{2} \implies [1, 1.5, -0.5, 2, 0.5]R1​=2R1​​⟹[1,1.5,−0.5,2,0.5]

  • Eliminasi R2: R2=R2−1R1  ⟹  [0,−3.5,2.5,−5,1.5]R_2 = R_2 - 1R_1 \implies [0, -3.5, 2.5, -5, 1.5]R2​=R2​−1R1​⟹[0,−3.5,2.5,−5,1.5]

  • Eliminasi R3: R3=R3−3R1  ⟹  [0,−3.5,2.5,−4,1.5]R_3 = R_3 - 3R_1 \implies [0, -3.5, 2.5, -4, 1.5]R3​=R3​−3R1​⟹[0,−3.5,2.5,−4,1.5]

  • Eliminasi R4: R4=R4−(−1)R1  ⟹  [0,5.5,0.5,0,4.5]R_4 = R_4 - (-1)R_1 \implies [0, 5.5, 0.5, 0, 4.5]R4​=R4​−(−1)R1​⟹[0,5.5,0.5,0,4.5]

3. Kolom 2 (Pivot R2)¶

  • Normalisasi R2: R2=R2−3.5  ⟹  [0,1,−0.71,1.43,−0.43]R_2 = \frac{R_2}{-3.5} \implies [0, 1, -0.71, 1.43, -0.43]R2​=−3.5R2​​⟹[0,1,−0.71,1.43,−0.43]

  • Eliminasi R3: R3=R3−(−3.5)R2  ⟹  [0,0,0,1,0]R_3 = R_3 - (-3.5)R_2 \implies [0, 0, 0, 1, 0]R3​=R3​−(−3.5)R2​⟹[0,0,0,1,0]

  • Eliminasi R4: R4=R4−5.5R2  ⟹  [0,0,4.43,−7.86,6.86]R_4 = R_4 - 5.5R_2 \implies [0, 0, 4.43, -7.86, 6.86]R4​=R4​−5.5R2​⟹[0,0,4.43,−7.86,6.86]

4. Kolom 3 (Pivot R3)¶

  • Tukar Baris: Karena A33A_{33}A33​ adalah 0, kita tukar R3R_3R3​ dengan R4R_4R4​:

    • R3↔R4R_3 \leftrightarrow R_4R3​↔R4​

  • Normalisasi R3: R3=R34.43  ⟹  [0,0,1,−1.77,1.55]R_3 = \frac{R_3}{4.43} \implies [0, 0, 1, -1.77, 1.55]R3​=4.43R3​​⟹[0,0,1,−1.77,1.55]

  • Eliminasi R4: R4R_4R4​ sudah memiliki 0 di kolom 3 (tidak ada perubahan).

5. Kolom 4 (Pivot R4)¶

  • Normalisasi R4: R4=R41  ⟹  [0,0,0,1,0]R_4 = \frac{R_4}{1} \implies [0, 0, 0, 1, 0]R4​=1R4​​⟹[0,0,0,1,0]

Setelah langkah-langkah di atas, matriks akan berbentuk Row Echelon Form (REF) dengan angka 1 di setiap diagonal utamanya. Kemudian, proses dilanjutkan dengan Eliminasi Mundur untuk menolkan elemen di atas pivot hingga menjadi Reduced Row Echelon Form (RREF).

import numpy as np

# Matriks Augmented untuk SPL 4 Variabel
A_original = np.array([
    [2, 3, -1, 4, 1],
    [1, -2, 2, -3, 2],
    [3, 1, 1, 2, 3],
    [-1, 4, 1, -2, 4]
], dtype=float)

def solve_gauss_to_ref(A):
    n = len(A)
    print("=== PROSES ELIMINASI GAUSSIAN (MENUJU REF) ===")
    print("Matriks Awal:")
    print(A)
    print("-" * 30)

    for i in range(n):
        # 1. Cek Pivot dan Tukar Baris jika perlu
        if abs(A[i, i]) < 1e-10:
            for k in range(i + 1, n):
                if abs(A[k, i]) > 1e-10:
                    A[[i, k]] = A[[k, i]]
                    print(f"Langkah {i+1}.0: Tukar Baris {i+1} dengan Baris {k+1}")
                    print(A)
                    break
        
        # 2. Normalisasi Baris (Membuat Utama 1)
        pivot = A[i, i]
        if abs(pivot) > 1e-10:
            A[i] = A[i] / pivot
            print(f"Langkah {i+1}.1: Normalisasi Baris {i+1} (R{i+1} / {pivot:.2f})")
            print(A)
        
        # 3. Eliminasi elemen di bawah pivot
        for j in range(i + 1, n):
            factor = A[j, i]
            if abs(factor) > 1e-10:
                A[j] = A[j] - factor * A[i]
                print(f"Langkah {i+1}.{j-i+1}: Eliminasi R{j+1} (R{j+1} - ({factor:.2f}) * R{i+1})")
                print(A)
        print("-" * 30)
    
    return A

def solve_ref_to_rref(A):
    n = len(A)
    print("\n=== PROSES ELIMINASI GAUSS-JORDAN (MENUJU RREF) ===")
    
    # Eliminasi elemen di ATAS pivot
    for i in range(n - 1, -1, -1):
        for j in range(i - 1, -1, -1):
            factor = A[j, i]
            if abs(factor) > 1e-10:
                A[j] = A[j] - factor * A[i]
                print(f"Langkah Mundur: Eliminasi R{j+1} (R{j+1} - ({factor:.2f}) * R{i+1})")
                print(A)
    print("-" * 30)
    return A

# Jalankan proses
matrix_working = A_original.copy()
matrix_ref = solve_gauss_to_ref(matrix_working)

print("\nHASIL AKHIR BENTUK REF (Row Echelon Form):")
print(matrix_ref)

matrix_rref = solve_ref_to_rref(matrix_ref.copy())

print("\nHASIL AKHIR BENTUK RREF (Reduced Row Echelon Form):")
print(matrix_rref)

# Solusi Akhir dari RREF (Kolom Terakhir)
solusi = matrix_rref[:, -1]
print("\nSolusi Akhir:")
variables = ['x1', 'x2', 'x3', 'x4']
for var, val in zip(variables, solusi):
    print(f"{var} = {val:.4f}")
