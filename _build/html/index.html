Project: Komputasi Aljabar Linear (KAL)

  • Nama: Muhammad Hafif Al-Firdaus

  • Nim: 230411100196

  • Kelas: KAL IF 2D

Proyek ini berisi kumpulan materi dan implementasi praktis untuk mata kuliah Komputasi Aljabar Linear. Fokus utama dari tugas ini adalah penyelesaian Sistem Persamaan Linear (SPL) menggunakan metode numerik yang diimplementasikan dalam Python.

Konten Proyek¶

Proyek ini terbagi menjadi beberapa bagian utama:

  1. Operasi Baris Elementer (OBE)

    • Penjelasan konsep dasar manipulasi baris matriks.

    • Implementasi langkah-demi-langkah menggunakan numpy.

    • Visualisasi interaktif melalui integrasi GeoGebra.

  2. Penyelesaian SPL 3 & 4 Variabel

    • Eliminasi Gaussian: Proses transformasi matriks augmented menjadi Row Echelon Form (REF).

    • Eliminasi Gauss-Jordan: Proses lanjutan untuk mencapai Reduced Row Echelon Form (RREF).

    • Substitusi Balik: Algoritma untuk ekstraksi nilai variabel dari bentuk REF.

Cara Menggunakan¶

Semua materi tersedia dalam format Jupyter Notebook (.ipynb) yang interaktif. Anda dapat menjalankan setiap sel kode untuk melihat proses eliminasi secara real-time termasuk detail normalisasi pivot dan eliminasi elemen di bawah/atas pivot.
